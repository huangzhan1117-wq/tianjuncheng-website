## 从零准备正式官网

### 1. 域名

建议优先选择短、好记、和公司名相关的域名。

可选方向：

- tianjuncheng.com
- hntjc.com
- hntjc.com.cn
- hntianjuncheng.com

准备方式：

- 在阿里云、腾讯云、华为云等平台购买域名
- 用公司营业执照完成域名实名认证
- 域名所有者建议与备案主体保持一致

### 2. ICP 备案

如果网站服务器放在中国大陆，必须先完成 ICP 备案后才能正式访问。

需要准备：

- 公司营业执照
- 法人身份证信息
- 网站负责人身份证信息和手机号
- 域名实名认证信息
- 服务器或静态托管资源
- 网站名称、网站用途、办公地址、联系电话
- 云服务商要求的核验材料

建议流程：

1. 先买域名并完成实名认证
2. 购买中国大陆服务器、云虚拟主机，或对象存储静态网站托管
3. 在同一家云服务商后台提交 ICP 备案
4. 备案通过后，把备案号添加到网站底部，并链接到工信部备案系统
5. 网站正式开通后，再按要求办理公安联网备案

注意：备案号不能提前编写。当前页面只能先放“备案办理后填写”。

### 3. 托管方式

当前网站是静态官网，推荐低成本、稳定的静态托管。

推荐方案：

- 阿里云 OSS 静态网站托管 + CDN
- 腾讯云 COS 静态网站托管 + CDN
- 云虚拟主机

如果你希望最省心，建议用“云虚拟主机”；如果希望加载速度和后期扩展更好，建议用“对象存储 + CDN”。

### 4. HTTPS 证书

正式域名上线后需要 HTTPS，微信分享也更推荐 HTTPS。

准备方式：

- 使用云服务商提供的免费 DV SSL 证书
- 或使用 CDN/托管平台自动签发的证书

### 5. 网站统计

需要你先注册统计平台，然后把统计代码发给我接入。

可选：

- 百度统计
- 友盟/CNZZ
- Google Analytics，适合海外统计

建议国内公司官网优先用百度统计。

### 6. 地图定位

需要选择地图服务并获取地图链接或 API Key。

可选：

- 高德地图
- 百度地图
- 腾讯地图

建议先用“地图分享/嵌入链接”，不需要复杂开发；如果要更精致的交互地图，再申请 API Key。

### 7. 在线表单

在线表单有两种方式。

简单方式：

- 金数据
- 腾讯问卷
- 飞书表单
- 企业微信表单

把表单链接接到官网按钮即可，维护成本低。

正式后台方式：

- 自建表单接口
- 数据保存到邮箱、表格或数据库
- 需要服务器、防垃圾提交和隐私提示

建议第一版先用金数据或腾讯问卷，快、稳、容易管理。
